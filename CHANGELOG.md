# Changelog

### Added
- Initial release of Wordle
- Kids mode: 3-letter words, 8 tries, 1 hint button
- Adult mode: 5-letter words, 6 tries, no hints
- On-screen keyboard with colour feedback
- Physical keyboard support
- Duplicate letter handling (correct algorithm)
- 180+ kids words, 340+ adult words bundled
- Win messages vary by number of tries used
- Dark mode via prefers-color-scheme
- About section: origin story + strategy tips
- GitHub Pages deployment
