# Contributing

## Ways to contribute

- Report bugs via [Issues](../../issues)
- Suggest new words for Kids or Adult word lists
- Suggest new features via [Issues](../../issues)

## Ground rules

- Single HTML file — no frameworks, no build step
- Vanilla JS only
- Test in both Kids and Adult modes
- Test on mobile (swipe + on-screen keyboard)
- Test in dark mode

## Pull Request checklist

- [ ] Tested in Chrome
- [ ] Tested on mobile
- [ ] Kids mode tested (3-letter words, hint works)
- [ ] Adult mode tested (5-letter words, no hint)
- [ ] Physical keyboard tested
- [ ] Dark mode tested
